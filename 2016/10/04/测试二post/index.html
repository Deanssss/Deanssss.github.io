小站的安家之路

  • web

“记得最初刚入计算机这个坑的时候就想过，以后要是能有一个属于自己的博客小站该多好，想写什么写什么，想怎么装扮怎么装扮，再也不用给某企鹅交票子了哈哈哈～到现在我发现建站原来是辣么简单。”

停停停停停！我说建站简单，是基于使用Hexo和Github这两大免费的好东西来说的，至于其它的，我也不会╮(╯▽╰)╭ ，以后研究到了再来分享咯。

(≖ ‿ ≖)✧好了，该来说说这个小站的建立了。

##准备好了吗？

1、准备好Git
2、准备好Node.js
3、准备好Github
4、准备好Hexo

###Git

####下载安装

  • Windows：下载并安装 git.
  • Mac ：使用 Homebrew, MacPorts 或下载 安装程序安装。
  • Linux (Ubuntu, Debian)：sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

    下面的用户名和邮箱需要和你的Github帐号保持一致。

###Node.js
下载Node.js。
直接安装即可。

###Github
进入Github官网。按步骤注册，验证账户。
完成后建立仓库(new repository|start a project)。Repository name填 Github用户名.github.io。

###Hexo
当上面的东西都准备好了之后，即可使用npm安装Hexo。
o(￣▽￣)ブWindows用户需要打开Git Bash。
